# Bankikuw - Website Builder Platform

منصة بناء المواقع - بديل لـ Wix

## المميزات الرئيسية

- 🎨 **Drag & Drop Editor** - محرر سحب وإفلات
- 🎯 **Component Library** - مكتبة مكونات جاهزة
- 📱 **Responsive Design** - تصاميم متجاوبة
- 🚀 **One-Click Publishing** - نشر بضغطة زر
- 👤 **User Dashboard** - لوحة تحكم المستخدم
- 💾 **Auto Save** - حفظ تلقائي

## التكنولوجيا

- **Frontend**: Next.js + React
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **DnD**: React Beautiful DnD
- **Language**: TypeScript

## البدء السريع

```bash
# تثبيت الحزم
npm install

# تشغيل على وضع التطوير
npm run dev

# البناء للإنتاج
npm run build

# بدء الخادم
npm start
```

الموقع سيكون متاح على: http://localhost:3000
