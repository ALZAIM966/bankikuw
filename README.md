# 🏗️ Bankikuw - منصة بناء المواقع الإلكترونية

منصة احترافية لإنشاء المواقع الإلكترونية بدون الحاجة إلى معرفة برمجية. اختر قالباً أو ابدأ من الصفر باستخدام محرر السحب والإفلات.

## ✨ المميزات

- 🎨 **محرر بصري متقدم** - سحب وإفلات سهل
- 📱 **متجاوب تماماً** - يعمل على جميع الأجهزة
- 🎯 **قوالب جاهزة** - متجر، مدونة، محفظة أعمال وأكثر
- 🌐 **نطاقات مخصصة** - ربط نطاقك الخاص
- 📊 **تحليلات** - تابع أداء موقعك
- 🔒 **آمن تماماً** - تشفير وحماية كاملة
- 💰 **رخيص** - خطط مرنة وبأسعار معقولة

## 🚀 البدء السريع

### المتطلبات
- Node.js 16+ 
- npm أو yarn

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/ALZAIM966/bankikuw.git
cd bankikuw

# تثبيت المكتبات
npm install

# نسخ متغيرات البيئة
cp .env.example .env.local

# تشغيل الخادم
npm run dev
```

سيتم فتح الموقع على: http://localhost:3000

## 🎯 الصفحات الرئيسية

### 🏠 الصفحة الرئيسية (`/`)
- عرض المميزات
- أسعار الخطط
- دعوة للتسجيل

### ✏️ محرر الموقع (`/editor`)
- محرر بصري متقدم
- سحب وإفلات للعناصر
- معاينة فورية
- حفظ تلقائي

### 👁️ معاينة الموقع (`/preview`)
- عرض الموقع كما سيبدو للزوار
- زر النشر المباشر

### 📝 التسجيل والدخول
- `/signup` - إنشاء حساب جديد
- `/login` - تسجيل الدخول

### 📊 لوحة التحكم (`/dashboard`)
- إدارة المواقع
- عرض الإحصائيات
- إنشاء موقع جديد
- تعديل المواقع الموجودة

### 🎨 القوالب الجاهزة (`/templates`)
- متجر إلكتروني
- مدونة شخصية
- محفظة أعمال
- موقع شركة
- موقع خدمات
- موقع تعليمي

### ⚙️ الإعدادات (`/settings`)
- إعدادات الملف الشخصي
- إدارة النطاق المخصص
- التفضيلات والإشعارات
- حذف الحساب

### ❓ المساعدة (`/help`)
- أسئلة شائعة
- دعم فني
- تواصل مباشر

## 🛠️ التقنيات المستخدمة

- **Frontend**: React, Next.js, TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Storage**: LocalStorage
- **Icons**: Unicode Emojis
